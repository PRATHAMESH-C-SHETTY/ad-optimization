🚀 TO DEPLOY ON NETLIFY (4 SIMPLE STEPS)

STEP 1: Set Up PostgreSQL Database (2 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Go to https://railway.app
2. Create new project → Provision PostgreSQL
3. Copy the database connection URL
4. Keep it safe for step 3

STEP 2: Push Code to GitHub (1 minute)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
git add .
git commit -m "Configure for Netlify deployment"
git push origin main

STEP 3: Deploy on Netlify (4 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Go to https://app.netlify.com
2. Click "Add new site" → "Import an existing project"
3. Select your GitHub repository
4. Configure build settings (or use defaults from netlify.toml):
   - Build command:     pip install -r requirements.txt && npm install
   - Publish directory: static
   - Functions:         netlify/functions

5. Click "Advanced" and add Environment Variables:
   ┌──────────────────────────────────────────────────┐
   │ FLASK_ENV     = production                       │
   │ FLASK_APP     = app.py                           │
   │ DATABASE_URL  = postgresql://... (from Railway)  │
   │ SECRET_KEY    = (generate below)                 │
   │ ENVIRONMENT   = production                       │
   │ PYTHON_VERSION= 3.11                             │
   └──────────────────────────────────────────────────┘

STEP 4: Generate Secret Key
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Run in your terminal:
  python -c "import secrets; print(secrets.token_hex(32))"

Copy the output and paste as SECRET_KEY in Netlify

STEP 5: Click Deploy and Wait!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
First build: 5-10 minutes (Python packages take time)
Future updates: 1-2 minutes
Status: Check Netlify dashboard

Your site will be live at: https://your-site-name.netlify.app

🎯 ENVIRONMENT VARIABLES REQUIRED

These must be set in Netlify dashboard:

FLASK_ENV=production
  → Tell Flask this is production environment

FLASK_APP=app.py
  → Flask application entry point

SECRET_KEY=<secure-random-key>
  → Cryptographic key for session management
  → Generate: python -c "import secrets; print(secrets.token_hex(32))"

DATABASE_URL=postgresql://user:password@host:port/dbname
  → Cloud database connection string from Railway
  → Format: postgresql://username:password@hostname:port/database_name

ENVIRONMENT=production
  → Custom environment indicator

PYTHON_VERSION=3.11
  → Explicit Python version for Netlify

Q: How much does it cost?
A: Netlify free tier is sufficient for most projects
   Railway provides free PostgreSQL tier
   Total cost: $0 (free tier) or pay-as-you-go

Q: What if something goes wrong?
A: Check NETLIFY_DEPLOYMENT_GUIDE.md troubleshooting section
   Review Netlify deployment logs
   Verify DATABASE_URL format
   Check environment variables are set

Q: Can I use a custom domain?
A: Yes! Set up in Netlify dashboard → Domain settings
   HTTPS works automatically!

Q: How do I update my application?
A: Push changes to GitHub main branch
   Netlify automatically deploys
   Check status in dashboard

Q: Where's my data stored?
A: PostgreSQL database in Railway
   Persists across deployments
   Automatic daily backups available

Q: Can I rollback to a previous version?
A: Yes, in Netlify dashboard → Deploys
   Click any previous deployment → Publish
